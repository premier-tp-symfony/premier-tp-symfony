# premier-tp-symfony

Commandes Git :
Jamais `PUSH` sur `master`
Jamais `merge` directement: `Pull Request` avant.

Récupérer :
```
git checkout master
git pull
Composer install
git checkout nom-branch
git rebase master
```

Pour push la branche :

```bash
git add .
git commit -m "Le commit"
git push origin labranche
```
